# 🚀 Exercícios em Java

Este repositório contém uma coleção de exercícios práticos desenvolvidos em Java, abordando conceitos fundamentais e avançados da linguagem. Ideal para estudantes e desenvolvedores que desejam praticar lógica de programação, orientação a objetos, estruturas de dados e outros tópicos.

## 📋 Pré-requisitos

Antes de começar, você precisará ter instalado em sua máquina:

- [Java Development Kit (JDK)](https://www.oracle.com/java/technologies/javase-downloads.html) versão 8 ou superior
- Uma IDE de sua preferência (Eclipse, IntelliJ, VS Code, etc.) ou um editor de texto simples

## 🔧 Como executar os exercícios

1. Clone este repositório:
   git clone https://github.com/seu-usuario/nome-do-repositorio.git

2. Navegue até a pasta do projeto:
   ```bash
   cd nome-do-repositorio
   ```

3. Compile e execute um arquivo específico (exemplo):
   ```bash
   javac Exercicio1.java
   java Exercicio1
   ```

   Ou, se estiver usando uma IDE, importe o projeto e execute a classe desejada.

## 🤝 Contribuição

Contribuições são bem-vindas! Se você deseja melhorar um exercício existente ou adicionar novos:

1. Faça um fork do projeto

4. Faça push para a branch (`git push origin feature/NovoExercicio`)
